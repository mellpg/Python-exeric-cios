#  Python Learning Repository

Este repositório reúne estudos e práticas em **Python** desenvolvidos por **Mel**, com foco no aprendizado gradual da linguagem e no fortalecimento da lógica de programação.

##  Objetivo

- Aprender e consolidar os fundamentos da linguagem Python  
- Praticar lógica de programação por meio de exercícios  
- Registrar a evolução técnica ao longo do aprendizado  

##  Conteúdo

O repositório inclui exemplos e exercícios relacionados a:

- Sintaxe básica do Python  
- Estruturas de controle  
- Funções  
- Estruturas de dados  
- Manipulação de arquivos
